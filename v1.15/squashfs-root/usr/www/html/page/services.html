You can set a service control list (SCL) to enable or disable services from being used.

Access Control -- Services

Interface
 
Service Enable Source Host(IP / Mask) :(Dst Port)

    
Helpful Hints...

If you wish to log in and manage your router from another Internet device, you can set the router to accept such commands from the Internet port. This option can be useful when your network administrator is not on site or the technical support requests such access.
