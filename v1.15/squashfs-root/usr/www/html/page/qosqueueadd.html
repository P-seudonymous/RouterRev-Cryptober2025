QUEUE CONFIG

In this page, you can configure a QoS queue entry and assign it to a specific network interface. Each QoS-enabled interface will have 3 queues by default. You can set precedence for each queue. The classifier uses the queue entry configured to handle the incoming packets accordingly.
Note: A smaller integer value of the precedence indicates higher priority of a queue.
Click "Save/Reboot" to save and activate the queue entry.
