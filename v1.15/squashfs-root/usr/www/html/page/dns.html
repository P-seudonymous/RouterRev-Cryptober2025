If you select "Obtain DNS server address automatically", the router uses the first DNS assignment obtained from one of the PPPoA, PPPoE, or MER/DHCP enabled PVCs, when the connection is established. If you select "Use the following DNS server addresses", enter the IP addresses of the primary and secondary DNS servers. The "Obtain DNS server address automatically" option is recommended unless the DNS servers do not work.
