Inbound IP Filtering

In this page, you can specify a filter name and at least one condition to create a filter for identify incoming IP traffic. All the specified conditions take effect simultaneously. Click "Apply" to save the filter and enable it.

Packets conform to the rule can be transmitted.

Helpful Hints...

Set the name of each rule.

A source IP address is a WAN address and a destination IP address is a LAN address.

By setting a rule, you can allow traffic from the WAN.

Select a time schedule for enabling the rule. If the list of schedules does not include your desired schedule, you can create one. The navigation path is as follows: Advanced -> Schedules

Click "Apply" to save the rule and it will be displayed in the Rules List.

Select the "Remove" check box corresponding to a rule in the Rules List and click the "Remove" button to permanently remove a rule.
