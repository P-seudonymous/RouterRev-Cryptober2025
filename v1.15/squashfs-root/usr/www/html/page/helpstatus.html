Device Info

It shows detailed information of the router, such as the firmware version, bootloader, and LAN IP address.

System Info
Model Name
Model of the ADSL router.
Time and Date
Internal time of the router.
Firmware Version
The current version of the firmware.

Internet Info
It shows WAN connections configured in the Setup -> Internet Setup page. The "Status" column shows whether the ADSL link is up, and it need not be configured.
Internet Connection
Settings of the created Internet connections.
Internet Connection Status
It indicates whether the selected connection is up or down.
Default Gateway
Current IP Address of the gateway. You can manually configure the gateway or set it to "auto" in the Advanced > Routing page..
Preferred and Alternate DNS

It shows the currently preferred (Primary) and alternate(Secondary) DNS servers. You can manually configure the DNS server information or set it to "auto" in theAdvanced > DNS page..

Upstream Line Rate(Kbps)
It indicates the upload rate in seconds. It is available only when you are connected to your ISP.
Downstream Line Rate(Kbps)
It indicates the download rate in seconds. It is available only when you are connected to your ISP.

Wireless Info
It shows the detailed WLAN settings configured in the Setup > Wireless page.
Mac Address
The wireless MAC Address.
Status
It indicates whether the wireless is enabled.
Network Name (SSID):
The wireless network name.
Visibility
It indicates whether the wireless network is visible to other wireless devices.
In the Visible mode, your wireless network name is broadcast to all the devices within the range of your signal. In the Invisible mode, you must manually enter the Wireless Network Name (SSID) on a wireless device in order to connect to the network.
Security Mode
It shows the current security settings.

Local Network Info
It shows the detailed LAN settings of the router.
Mac Address
The MAC address of the router.
IP Address
The IP Address of the router.
Subnet Mask
The subnet mask of the router.
DHCP Server
It indicates whether DHCP is enabled on the router.

Wireless Clients

Displays the stations (identified by their BSSIDs) that are associated with your wireless router. Click "Refresh" to refresh the page for displaying new wireless stations.

DHCP Clients

It shows the computers that obtained IP addresses from the DHCP server but the lease time of whose IP address elapsed, each identified by host name and MAC address.

IPv6 Status

In this section you can see the information for the IPv6 Connection.

IPv6 Connection

Wan Connection : the name of WAN connection.

Connection Type : the type of WAN connection.

IPv6 Address/Prefix Len : the address and prefix of IPV6 connection.

Gateway : IPV6 Gateway.

Pri Dns : the First DNS address.

Pri Dns : the Second DNS address.

Prefix Info : the information of IPV6 prefix.

Status : the status of WAN connection.

Logs

If the log is enabled, the system starts to log the selected levels of events, which may include Emergency, Alert, Critical, Error, Warning, Notice, Informational, and Debugging. Events that are above or equal to the selected display level will be displayed.

The navigation path to configure the system logs is as follows: Maintenance > System Log.

Firewall Logs

In this page, you can view firewall logs.

Statistics

The "Statistics" page displays the transmit and receive data of the LAN, WAN, and wireless packets.

Transmitted
The number of packets sent from the router.
Received
The number of packets received by the router.
Transmitted Packets Dropped
The number of packets that are dropped while being sent, due to errors, collisions, or router resource limitations.
Received Packets Dropped
The number of packets that are dropped while being received, due to errors, collisions, or router resource limitations.
Errs
The number of transmission failures that cause loss of a packet. A noisy radio-frequency environment can cause a high error rate on the wireless LAN.
ADSL Statistics
It shows ADSL statistics data, including link status, downstream and upstream rates, type, and line coding.

Route Info

The Route Info page shows the routing information, including IP addresses of the destination, gateway, and subnet mask.
