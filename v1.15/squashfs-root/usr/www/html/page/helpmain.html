Ipv6

In this page you can open ipv6 function. If you choose to enable, the device will support ipv6 Internet access. If you choose to turn off, the device will not support ipv6 Internet access

System

In this page, you can manage the configuration of the router, reboot the router, back up the current configuration, load previously saved settings, and restore the router to factory default settings. Restoring the router to factory defaults will erase all the current settings, including the created rules.
Reboot
This function is to restart the router for the settings to take effect.
Back Up Settings
You can use this function to save the configuration in a file on the computer for later use. Be sure to save the configuration before updating the firmware.
Update Settings
You can use this function to restore the settings of the router previously saved.
Restore Default Settings
This function is to restore all the settings to the settings configured when the router was shipped from the factory. All current settings will be lost. If you want to save the current settings, please click Back Up Settings.

Firmware Update

In the "Firmware Update" page, you can update the firmware of the router to the latest, to improve the functionality and performance.

To update the firmware, do as follows:

  1. Click Browseto locate the update file on your computer.
  2. Select the file and click Update Firmware to start updating the firmware. This process takes about a minute.
  3. Wait for the router to reboot. The rebooting process takes about a minute.
Note: Some firmware updates reset the settings of the router to factory defaults. Before performing an update, be sure to save the current settings. The navigation path is as follows: Maintenance -> System .

Access Controls

Account Password

In this page, you can set a password for accessing the Web management interface. By default, no password is configured. You are recommended to create a password to keep your new router secure.

You can access the DSL router as one of the three user accounts: admin, support, and user.

admin Password
Enter a password for the "admin" user, who has full access to the Web management interface.
support Password
The "support" user can be used by an ISP technician to access the DSL router for maintenance and to run diagnosis.
user Password
The "user" user can view settings and statistics, as well as update the firmware of the router.
Services
Allows access to the router via FTP, HTTP, ICMP, SNMP, SSH, TELNET, and TFTP.
LAN
Accesses the DSL router from the local network (LAN) side.
WAN
Accesses the DSL router from the ISP side.
IP Address
If this option is enabled, only specified IP addresses can access the router. This option works with Services.

Diagnostics

Diagnostics runs diagnostic tests to check the DSL connection. The results are about the following three connections:

  • Connection to the local network
  • Connection to the DSL service provider
  • Connection to the Internet service provider

System Log

If the log is enabled, the system starts to log all selected events.

Log Level
The level of logs to be saved.
Display Level
The level of logs to be displayed in the log page.

If the mode is set to "Remote", logs are sent to the specified IP address and UDP port of a remote syslog server. If the mode is set to "Local", logs are recorded in the local computer. If you set "Both", logs are saved in both the local computer and the remote syslog server. Select the desired values and click "Apply" to save the system log settings.

View System Log
In this page, you can view logs corresponds with the level you set.
