Schedule work with a number of other features to define the time period that those features take effect.

Set the name of each rule. For example, a schedule that lasts from 3:00pm to 9:00pm every Monday to Friday can be named "After School".

NOTE: Setup -> Time and Date must be set for schedule to work properly.
