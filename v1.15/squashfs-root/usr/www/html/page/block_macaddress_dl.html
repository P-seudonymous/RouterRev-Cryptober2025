Block MAC Address

This page used to control a special LAN device access to the router. It is based on the MAC to identify the device and MAC is the exclusive ID for every device. The LAN device can be found under DHCP client list. User can configure maximum of 20 rules.

note : If you select"Enable Block MAC, allow following PC to access network",please put your PC MAC also adds to the rules.Otherwise,the PC will not be able to access cpe!
